Completed Projects:

A Crop Type Prediction App

I created a web app using python and Streamlit which uses a random forest model to predict which crop types can be grown in the field specified. The app uses the Google Earth Engine API to get various soil, climate and topographical data for the field. The app also shows a rainfall and temperature forecast for the week ahead for the field. In addition, NDVI for a certain year for a specific field can be displayed. Lastly, the collected data is fed in to LLama 13B which provides AI feedback on the field values.

Land Use Land Cover of Kilala, Kenya

I created and ran two machine learning techniques for a farming region near the town of Kilala, a major maize growing region in Kenya. Here I used QGIS to create training input data and validation data and then used R Studio to run the SVM and RF machine learning models and to calculate the results based on the validation dataset.

Predicting Machine Failure

The goal of the project was to detect a pump failure. The typical air pump failure is a temporary pressure decrease in the first half of the cycle. As part of a company assignment, I demonstrated my exploratory data analysis skills, feature engineering, and machine learning skills by creating a model to predict pump failure. My model achieved an an F1 Score of 0.913 and Recall of 0.924. The Jupyter Notebook can be found on my GitHub.

Predicting House Prices

During this side project I sought to further my python, data science and MLops skills by scraping the websites of popular house rental agencies. The data was collected, cleaned and used to train a CatBoost model to predict rental prices. This model can be useful for renters to identify if they are being charged a fair price. The model works for the 20 most popular neighbourhoods in South Africa.
